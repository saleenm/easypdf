# EasyPDF — أدوات PDF المجانية

موقع أدوات PDF مجاني يعمل مباشرة في المتصفح بدون رفع الملفات لأي سيرفر.

## الأدوات المتاحة
- 🔗 دمج PDF
- ✂️ تقسيم PDF
- 🔄 تدوير الصفحات
- 🗜️ ضغط PDF
- 💧 علامة مائية
- 🔢 ترقيم الصفحات
- 🖼️ PDF إلى صور
- 📄 PDF إلى نص
- 📸 صور إلى PDF

## الرفع على GitHub Pages
1. ارفع المجلد على GitHub
2. اذهب إلى Settings → Pages
3. اختر Branch: main / root
4. الموقع سيكون جاهزاً على: `https://username.github.io/easypdf`
